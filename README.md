# Silky's Blog ✷

A storytelling website for short stories, incidents, and reflections — designed in a hand-made **collage / journal zine** aesthetic (graph-paper, marker headlines, doodles, drifting song lyrics).

**Live:** https://darshan-1820.github.io/Silkys_blog/

## Features
- Collage/scrapbook design — magenta + cobalt marker palette, hand-drawn doodles, taped polaroid story cards.
- Day / night reading modes (researched, accessible dark theme).
- Hand-drawn radio with lyrics drifting out; journal-style reading view with drop caps & pull-quotes.
- Interactive touches: a mouse marker-trail, an eye whose iris follows the cursor, and — at night — fireflies that scatter and an owl that startles or flies away.
- Reading UX: progress bar, read-time, "keep reading" rail, fully responsive (MacBook + phones).

## Tech
Single self-contained `index.html` (HTML + CSS + vanilla JS, Google Fonts). No build step. Hosted on GitHub Pages.

## Status
Design prototype with placeholder content — pending the final name, real photos, and a CMS so stories can be published without code.
